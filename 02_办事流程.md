# 办事流程指南

**维护信息**：最后更新时间 2026年7月14日 | 责任部门：学校办公室

---

## 在校生办事流程

### 课程注册

每学期开学前一周，学生需登录学校教务系统进行课程注册。具体步骤如下：

1. **登录教务系统**：使用学号和密码登录学校教务管理系统
2. **查看课表**：在"课程管理"栏目中查看本学期课程安排
3. **确认注册**：确认课程信息无误后，点击"确认注册"按钮
4. **打印课表**：注册成功后，可打印课表作为参考

⚠️ **注意**：未按时完成课程注册的学生将无法参加课程学习和考试。

📝 **提示**：如有课程冲突或需要调整，需在注册截止日期前联系学院教学秘书。

### 成绩查询

学生可通过以下方式查询成绩：

1. **在线查询**：登录教务系统，在"成绩查询"栏目中查看各科成绩
2. **成绩单打印**：在教务系统中下载并打印成绩单
3. **咨询辅导员**：如对成绩有疑问，可联系学院辅导员咨询

⚠️ **注意**：成绩公布后，如对成绩有异议，需在一周内提出复查申请。

### 请假流程

学生因特殊原因需要请假时，需遵循以下流程：

1. **填写请假单**：在学院办公室领取或在教务系统下载请假单
2. **提交申请**：填写请假单后，提交给辅导员审核
3. **审批通过**：辅导员审核通过后，请假单生效
4. **销假**：假期结束后，需及时到学院办公室销假

⚠️ **注意**：请假天数超过3天的，需经学院领导批准；超过7天的，需经教务处批准。

### 学生证补办

学生证遗失或损坏时，需按照以下流程补办：

1. **挂失登记**：先到学生工作办公室进行挂失登记
2. **填写申请表**：领取并填写《学生证补办申请表》
3. **提交材料**：提交申请表和一寸照片1张
4. **缴纳费用**：缴纳补办费用（20元/张）
5. **领取新证**：一般在3个工作日后可领取新证

📝 **提示**：补办期间可使用临时学生证，临时学生证在学生工作办公室办理。

### 图书馆相关

#### 借书流程

1. **校园卡绑定**：首次借书需携带校园卡到图书馆服务台进行绑定
2. **检索书籍**：使用图书馆网站或馆内检索机查找所需书籍
3. **借阅手续**：携带书籍和校园卡到借阅台办理借阅手续
4. **借阅期限**：每本书借阅期限为30天，可续借一次（续借期限15天）

⚠️ **注意**：逾期未还的书籍将产生滞纳金（0.1元/天/本）。

#### 座位预约

1. **登录系统**：登录图书馆座位预约系统
2. **选择座位**：选择日期、时间段和座位
3. **确认预约**：确认预约信息后提交
4. **签到入座**：在预约时间前15分钟内到馆签到

📝 **提示**：未按时签到的座位将被释放，连续3次违约将限制预约权限。

### 校园卡服务

#### 校园卡充值

1. **自助充值**：在各教学楼、食堂的自助充值机上使用银行卡充值
2. **线上充值**：登录校园服务平台，使用微信或支付宝充值
3. **人工充值**：到校园卡服务中心使用现金充值

#### 校园卡挂失

1. **自助挂失**：在自助充值机上或通过校园服务平台进行挂失
2. **人工挂失**：携带身份证到校园卡服务中心办理挂失

⚠️ **注意**：挂失前发生的消费由持卡人自行承担。

#### 校园卡补办

1. **挂失登记**：先进行挂失登记
2. **提交材料**：携带身份证到校园卡服务中心
3. **缴纳费用**：缴纳补办费用（20元/张）
4. **领取新卡**：一般在3个工作日后可领取新卡

### 实习与就业

#### 实习申请

1. **选择实习单位**：学生自主选择实习单位或参加学校组织的实习双选会
2. **填写申请表**：填写《学生实习申请表》，经学院审核同意
3. **签订协议**：与实习单位签订实习协议
4. **报备审批**：将实习协议提交学院备案

⚠️ **注意**：实习期间需遵守实习单位的规章制度，认真完成实习任务。

#### 就业指导

学校就业指导中心为学生提供以下服务：
- 就业信息发布
- 简历制作指导
- 面试技巧培训
- 职业规划咨询
- 校园招聘会组织

📝 **提示**：建议学生尽早关注就业信息，提前做好就业准备。

## 教师常用流程

### 课程安排

每学期末，教师需按照以下流程进行下学期课程安排：

1. **登录教务系统**：使用教师账号登录教务管理系统
2. **查看教学任务**：在"教学任务"栏目中查看下学期教学任务安排
3. **确认课程**：确认课程信息无误后，点击"确认"按钮
4. **提交课表**：根据教学任务安排，提交个人课程表

⚠️ **注意**：如需调整课程安排，需在规定时间内联系学院教学秘书。

### 课件上传

教师可通过以下方式上传课件：

1. **登录教学平台**：使用教师账号登录学校在线教学平台
2. **选择课程**：在"我的课程"中选择对应的课程
3. **上传课件**：在课程资源栏目中上传课件文件
4. **发布通知**：可发布课程通知，提醒学生查看课件

📝 **提示**：建议提前上传课件，方便学生预习和复习。

### 成绩录入

考试结束后，教师需按照以下流程录入成绩：

1. **登录教务系统**：使用教师账号登录教务管理系统
2. **选择课程**：在"成绩管理"中选择对应的课程
3. **录入成绩**：按照学生学号顺序录入成绩
4. **提交审核**：确认成绩无误后，提交成绩进行审核

⚠️ **注意**：成绩录入有严格的时间限制，需在规定时间内完成。

### 教室预约

教师因教学或科研需要使用教室时，需按照以下流程预约：

1. **登录预约系统**：使用教师账号登录教室预约系统
2. **选择教室**：选择日期、时间段和教室类型
3. **填写用途**：填写使用用途和人数
4. **提交申请**：提交预约申请，等待审核

📝 **提示**：热门教室建议提前一周预约，以免影响教学安排。

### 科研项目申报

教师申报科研项目时，需按照以下流程进行：

1. **查看通知**：关注学校科研处发布的项目申报通知
2. **准备材料**：根据通知要求准备申报材料
3. **提交申请**：在规定时间内提交申报材料
4. **审核评审**：学校组织专家进行审核和评审
5. **立项公示**：评审通过后，进行立项公示

⚠️ **注意**：申报材料需真实准确，如有弄虚作假将取消申报资格。

### 教师培训

学校为教师提供以下培训机会：

1. **岗前培训**：新入职教师需参加岗前培训
2. **业务培训**：定期组织教学业务培训
3. **学术交流**：组织学术研讨会和学术讲座
4. **出国进修**：提供出国进修和访问学者机会

📝 **提示**：教师可根据自身需求选择参加相关培训。

### 办公用品领用

教师需领用办公用品时，按照以下流程进行：

1. **填写领用单**：在学院办公室领取《办公用品领用单》
2. **填写内容**：填写所需办公用品名称和数量
3. **审批签字**：经学院领导审批签字
4. **领取物品**：携带领用单到后勤服务中心领取物品

### 会议室预约

教师因会议需要使用会议室时，按照以下流程预约：

1. **登录预约系统**：使用教师账号登录会议室预约系统
2. **选择会议室**：选择日期、时间段和会议室
3. **填写信息**：填写会议主题和参会人数
4. **提交申请**：提交预约申请，等待审核

📝 **提示**：大型会议室需提前两天预约。
